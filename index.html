Leaflet Giphy API Demo

The example below shows the usage of an experimental TileStache provider class, allowing a user to create tiles based on results from a query to the Giphy API.

More information about the provider class:https://github.com/mattmakesmaps/TileStache/blob/master/TileStache/Goodies/Providers/GiphyAPI.py

Note, Non-Shark cartography is derived from MapBox's Geography Class Style.
